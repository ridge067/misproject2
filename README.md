# MySite — Student Portfolio Scaffold

This repository contains a multi-page static website scaffold for a student portfolio. Replace the placeholder content (images, videos, resume) with your own.

Files added:
- `index.html` — Home page with headshot, hobby summaries, and welcome video placeholder
- `hobbies.html` — Hobbies details page
- `discover.html` — UMD & Duluth page with video placeholder
- `resume.html` — Web-formatted resume and PDF download link
- `career.html` — Career interests page
- `game.html` — Interactive memory matching game
- `styles.css` — Central styles
- `script.js` — Shared JS: theme toggle, nav, and game helper
- `assets/` — Placeholders for images, videos, resume PDF (see `assets/README.txt`)

How to view locally:
1. Open `index.html` in your browser (double-click or right-click -> Open with).
2. Replace files in `assets/` with your images and videos.

Notes & next steps:
- Update content in each HTML file with your personal text and contact info.
- Replace placeholder media in `assets/`.
- You can host this as a static site via GitHub Pages or any static hosting provider.
